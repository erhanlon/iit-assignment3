Tools & Technologies

Software

The softwares required to develop the system are Visual Studio Code to write the code for the system, Python programming language for the back end, and a MySQL database to store the data. For the front end, HTML will be used to display the input and output elements. To make the data flow diagrams and entity relation diagrams to document the work for future reference, LucidChart will be used. For the group work and combining the work, GitHub will be used.

Hardware

1 GB or higher RAM for the server, which will be Apache 2.4.23. NFC technology will be used to enable contactless payment. Monitors will be needed to display the output on the screen, which will have tactile input for the user commands. No further hardware is required.
