Tools & Technologies

Software

The softwares required to develop the system are visual studio code to write the code for system, python programming laguage will be used in backend along with MySql database to store the data. For front end HTMl will be used as markup language to display the input and output elements. To make data flow diagrams and entity relation diagram to document the work for future reference lucid chart will be used. For the group work and combining teh work GitHub will be used.

Hardware

1 GB orr higher RAM on Server, programming server will be Apache 2.4.23. NFC technology will be used to enable the contactless payment. Any monitor to display the outputs on the screen along with keyboard, and mouse as peripheral devices for input the user commands. Other than this, no other hardwares are required.
